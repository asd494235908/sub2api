# Design System Document

This document outlines the core design principles and specifications for our user interfaces, ensuring consistency and brand cohesion across all products.

## Theme Overview

Our design system utilizes a **dark color mode**, providing a modern and comfortable viewing experience, especially in low-light environments. The overall aesthetic is driven by a focus on clarity and user engagement.

## Color Palette

The color palette is built around a set of distinct colors that define our brand's visual identity.

-   **Primary Color**: `#1E40AF` - A strong, deep blue used for key interactive elements, calls to action, and brand accents, conveying trust and reliability.
-   **Secondary Color**: `#64748B` - A sophisticated, muted gray-blue that supports the primary color, suitable for less prominent UI elements, data visualizations, and secondary information.
-   **Tertiary Color**: `#0F172A` - A very dark blue-gray, almost black, serving as an accent for highlights, badges, or decorative elements, providing depth and contrast.
-   **Neutral Color**: `#F8FAFC` - A light, almost white, base color, effectively used for backgrounds, surfaces, and non-chromatic elements even in dark mode context where it might be used for foreground elements or specific card backgrounds to create contrast.

**Named Colors for Specific Roles:**
- `primary`: `#1E40AF`
- `secondary`: `#64748B`
- `tertiary`: `#0F172A`
- `neutral`: `#F8FAFC`

## Typography

We leverage a precise typographic hierarchy to ensure readability and visual harmony.

-   **Headlines**: `manrope` - Used for all major titles and headings, providing a contemporary and clean look.
-   **Body Text**: `inter` - Chosen for its excellent readability across various screen sizes and contexts, ideal for paragraphs and detailed information.
-   **Labels**: `inter` - Consistent with body text, ensuring clarity and familiarity for all interactive labels and small text elements.

## Shape and Form

Our UI elements feature **subtly rounded corners (roundedness level 1)**. This approach provides a soft, approachable feel while maintaining a sense of modernity and crispness, avoiding overly sharp or completely pill-shaped forms.

## Spacing and Layout

We adopt a **normal level of spacing (spacing level 2)** within and between UI elements. This ensures a balanced visual density, providing enough whitespace for clarity without feeling overly sparse or cramped, enhancing readability and user focus.
